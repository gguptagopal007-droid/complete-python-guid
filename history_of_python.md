# History of Python

Python was developed by Guido van Rossum in the late 1980s.

## Timeline

### 1989
Work on Python started.

### 1991
Python 0.9.0 was released.

### 2000
Python 2.0 was released.

### 2008
Python 3.0 was released.

### Present
Python is one of the most widely used programming languages.

## Conclusion

Python has evolved continuously while maintaining simplicity and readability.
