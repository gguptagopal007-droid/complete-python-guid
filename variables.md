# Variables in Python

A variable is a container used to store data.

## Example

```python
name = "Gopal"
age = 45
```

## Rules for Naming Variables

- Must start with a letter or underscore
- Cannot start with a number
- Cannot contain spaces
- Case-sensitive

## Examples

Valid:
- name
- age
- student_name

Invalid:
- 1name
- student name
```
