# Elif Statement in Python

## What is Elif?

The elif statement is used when we need to check multiple conditions.

## Syntax

```python
if condition1:
    statement
elif condition2:
    statement
else:
    statement

marks = 75

if marks >= 90:
    print("Grade A")
elif marks >= 75:
    print("Grade B")
else:
    print("Grade C")

Advantages
Handles multiple conditions
Improves readability
Reduces nested if statements
Summary

Elif allows a program to make multiple decisions efficiently.
