# String Methods in Python

## upper()

Converts text to uppercase.

```python
name.upper()
```

## lower()

Converts text to lowercase.

```python
name.lower()
```

## title()

Converts first letter of each word to uppercase.

```python
name.title()
```

## replace()

Replaces a word or character.

```python
name.replace("Gopal","Rahul")
```

## len()

Returns length of string.

```python
len(name)
```
