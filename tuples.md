# Tuples in Python

## What is a Tuple?

A Tuple is a collection of ordered items.

Tuples are similar to lists but cannot be changed after creation.

## Creating a Tuple

```python
fruits = ("Apple", "Banana", "Mango")
```

## Accessing Elements

```python
print(fruits[0])
```

## Tuple Characteristics

- Ordered
- Immutable
- Allows duplicate values

## Tuple Methods

### count()

Returns the number of times a value appears.

### index()

Returns the index of a value.

## Example

```python
numbers = (10, 20, 30)

print(numbers[1])
```
