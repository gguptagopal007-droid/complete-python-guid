# Installing Python

## Step 1

Visit the official Python website:

https://www.python.org

## Step 2

Download the latest version of Python.

## Step 3

Run the installer.

## Step 4

Check:

☑ Add Python to PATH

## Step 5

Click Install Now.

## Verify Installation

Open Command Prompt and type:

python --version

Example Output:

Python 3.13.0
